## Frequently Asked Questions (FAQ)

**What makes Lunacy different from alternatives?**
Lunacy combines graphic design and vector illustration with deep macOS integration, providing Sketch-compatible design tool through optimized performance and native system features.

**Is Lunacy compatible with Apple Silicon?**
Yes, Lunacy is built as a universal binary supporting both Intel and Apple Silicon processors with optimized performance for each architecture.
